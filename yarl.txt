#MyPeriyar உலகின் மூத்த மொழி தமிழ் எனும் போது, தமிழுக்குக் கடவுள் இருந்தால் அதன் பெயர் தமிழில் இருக்கும் அல்லவா? ( விடுதலை - 03.11.1971 )
#MyPeriyar நம்மை அவ்வளவு நாசப்படுத்தவே வடநாட்டு ஏகாதிபத்தியம் தயார் செய்து கொண்டிருக்கிறது. சுயராஜ்யம் என்ற பேரால். ( விடுதலை ~ 27.07.1947 )
#MyPeriyar எதைச் சொன்னால் மக்கள் ஏமாறுவார்களோ அதைக் கண்டுபிடித்துச் சொல்லுவார்கள். கலவரம் உண்டாக்கியே பிழைக்கக் கூடியவர்கள். இவர்களைப் போன்றவர்களைப் பெரிதும் கொண்டது தான் அரசியல் கட்சிகள். இதைப் பித்தலாட்ட மோசடி ஸ்தாபனம் என்றே சொல்ல வேண்டும். ( விடுதலை - 04.10.1952 )
#MyPeriyar பூரண சுதந்திரம் என்பது நம் நாட்டிலுள்ள சாதிமத வித்தியாசங்கள் நீங்கினால்தான், கிடைத்த சுதந்திரத்தை மக்கள் நலத்திற்கும், தேசநலத்திற்கும் பயன்படுத்த முடியமேயொழிய இந்த நிலைமையில் அவனவன் சுயநலத்திற்கும், சுயசாதி நலத்திற்கும்தான் உபயோகிப்பான்”(குடியரசு ~ 14.09.1930)
#MyPeriyar மதம் மக்களை ஒன்றுபடுத்த ஏற்பாடு செய்யப்பட்டது என்றால், ஒரு மதத்தைச் சார்ந்தவர் இன்னொரு மதத்தாரை வெறுக்கவும், இழிவுப்படுத்தவும் காரணம் என்ன?( விடுதலை - 07.07.1971) 
#MyPeriyar கல்வி கற்பதில் மதத்தின் ஆதிக்கம் வந்து புகுந்து - அறிவுக்காகக் கல்வி என்பது மாறி - முட்டாள்தனமும், விசாரணையற்ற தன்மையும் வருவதற்கே கல்வி பயன்படும்படி ஆகிறது. ( குடிஅரசு - 27.09.1931 )
#MyPeriyar இளைத்தவனை வலுத்தவன் கொடுமைப்படுத்துவதும்; ஏமாந்தவனைத் தந்திரசாலி ஏமாற்றுவதும் இன்று ஆஸ்திகர்களுடைய கடவுள்களின் இரண்டு கண்களாகவும்; தேச பக்தர்கள், தேசாபிமானிகள் என்பவர்களின் ஜீவநாடியாகவும் இருந்துவருகின்றன.( குடிஅரசு - 29.09.1935 )
#MyPeriyar வள்ளுவரும் ஒரு நாத்திகராகவே பல இடங்களில் நடந்து கொள்கின்றார். புத்தருக்குப் பிறகு, வள்ளுவரைத் தான் நாத்திகர் என்று கூற வேண்டும்.  (பெரியார், விடுதலை - 22.03.1960)
#MyPeriyar வருணாச்சிரமதருமக்காரர்கள் சனாதன தருமம் காப்பாற்றப்பட வேண்டும் என்று சொல்லுவதற்கும் சுயராஜ்யத்தைவிட சனாதன தருமமும், வருணாச்சிரம தருமமும் காப்பாற்றப்படுவதே முக்கியமானதென்று சொல்லுவதற்கும் சமஸ்கிருதப் பாஷைப் படிப்பும் அதில் உள்ள நூல்களுமே காரணமாகும். ( குடி அரசு  – 17.01.1932 )
#MyPeriyar எந்த மனிதருக்கும் அவரது அபிப்பிராயம் என்னும் பேரால் எதையும் சொல்ல உரிமையுண்டு. அதைத் தடுப்பது என்பது யோக்கியமற்ற காரியமாகும். யார் எதைச் சொன்னாலும், சொல்ல அவகாசமளித்த பிறகுதான் அதை மறுத்துப் பேச வேண்டும். (பெரியார், குடி அரசு - 20.08.1933)
#MyPeriyar கடவுள் சொன்னது; மகான் சொன்னது; ரிஷி சொன்னது; அவதாரப் புருசர்கள் சொன்னது என்று பார்க்கிறார்களே ஒழிய தன் புத்தி என்ன சொல்கிறது என்று பார்ப்பதில்லை. ( விடுதலை - 03.04.1961 )
#MyPeriyar "நம்நாட்டில் பத்திரிகைகள் ஏராளமாக இருந்தும்கூட அவை எல்லாம் மக்களை ஏய்த்து, வயிறு வளர்ப்பதற்காகவும், அவரவர்கள் சுயநலத்துக்கு ஆகவுமே உள்ளவையாக இருக்கின்றன..." (விடுதலை" 19-10-1958)
#MyPeriyar பத்திரிக்கைகாரர்கள்  மனது வைத்தால் நாட்டைத் திருத்தலாம்; மேல் நிலைக்கும் கொண்டு வரலாம். ஆனால் இந்த நாட்டுப் பத்திரிக்கைகள் எல்லாம் வயிற்றுப் பிழைப்புக்காகப் பலரால் நடத்தப்படுகிறது. ( விடுதலை - 30.10.1960 )
#MyPeriyar ஆரிய விஷம் உடலில் ஊறிப் போயிருக்கிறது. அம்மயக்கத்திலிருந்து தெளிய பகுத்தறிவு என்ற 'இஞ்சக்ஷன்'கொடுக்க வேண்டும். (பெரியார், குடி அரசு - 10.02.1945)
#MyPeriyar எந்த மதத்தாலாவது, அதைச் சேர்ந்த உலகமக்கள் வாழ்க்கையிலோ பொருளாதாரத்திலோ சுதந்திரமோ, சமத்துவமோ பெற்று, கேவலம் ஜீவனத்திற்காக மற்ற மனிதனுக்கு அடிமையாகாமல் வாழ்வதற்கு இடமளித்திருப்பதாய் காணமுடியவில்லை. ( குடி அரசு ~ 20.11.1932 )
#MyPeriyar இந்த உலகத்தில் உள்ள எல்லா அகராதிகளையும் கொண்டு வந்து போட்டு ஏடு ஏடாகப் புரட்டிப் பார்த்தாலும் அழகும், பொருளும், சக்தியும் நிறைந்த வார்த்தை யாகிய சுயமரியாதை என்ற வார்த் தைக்கு மேலானதாகவோ ஈடானதாகவோ உள்ள வேறு ஒரு வார்த்தையை யாராலும் கண்டு பிடிக்க முடியாது.( குடிஅரசு ~ 1.6.1930 )
#MyPeriyar மனிதனைச் சீர்திருத்தி முன்னுக்கு கொண்டு வர வேண்டும் என எண்ணுபவர்கள் மிக மிகக் குறைவு. அறிவின் பெருமையை உணர்வதற்குச் சிலர் முதுகைப் பார்த்துத் தான் பழகி இருக்கிறார்களே ஒழிய, முன்னே பார்த்துப் பழகவில்லை!(பெரியார், விடுதலை - 04.04.1961)
#MyPeriyar எப்போதும் நான் நன்றி பெறுவதில் கவலைபடுவதே இல்லை. மனித ஜீவனிடம் நன்றி எதிர்பார்ப்பது அறிவில்லாத தன்மையாகும். ( விடுதலை ~  29.03.1967 )
#MyPeriyar பெண் அடிமை என்பது மனித சமூக அழிவு என்பதை நாம் நினைக்காததாலேயே, வளர்ச்சி பெற வேண்டிய மனித சமூகம் பகுத்தறிவு இருந்தும் நாள்தோறும் தேய்ந்து கொண்டே வருகிறது” ( குடிஅரசு’ 16-06-1935 )
#MyPeriyar பெண்கள் சுதந்திரத்துக்கும், பெண்கள் விடுதலைக்கும் அவர்கள் மனப்பான்மை சற்று மாறியேயாக வேண்டும். நான் அடிமையாய்த் தான் இருப்பேன். நீ மாத்திரம் எனக்கு எஜமானாய் இருக்கக் கூடாது என்பதில் அர்த்தமேயில்லை”( குடிஅரசு 21.06.1931 )
#MyPeriyar எதையும் உங்கள் பகுத்தறிவு கண் கொண்டு பாருங்கள். நான் கூறுவதைக்கூட ஏதோ ஒரு ரேடியோ நிகழ்ச்சியைக் கேட்பது போல் கேளுங்கள்; ஆனால் கவனத்தோடு கேளுங்கள்! ( விடுதலை ~ 05.10.1948 ) 
#MyPeriyar எல்லாம் கடவுள்செயல், எங்கும் கடவுள் ஆணை நடக்கிறது என்பவர்களிடத்தில் நமக்குப் பேச்சில்லை. ஆனால் அதையும் சொல்லிக்கொண்டு உங்களையும் ஈடேற்றுவதாக வருகிறார்களே, அந்தப் புரட்டர்களுக்கு ஏமாந்து விடாதீர்கள்( குடிஅரசு - 10-03-1945 ) 
#MyPeriyar என்னைப் போல ஒரு சாதாரண மனிதன் பேசுவதாலே உங்கள் கடவுள், உங்கள் சமயம் போய்விடும் என்று பீதி அடைவீர்களேயானால் உங்கள் கடவுளுக்கும், உங்கள் மதத்திற்கும் உள்ள யோக்கியதை தான் என்ன? 
#MyPeriyar கடவுள் கண்ணுக்கு எட்டாதவன், அறிவுக்கு எட்டாதவன், மனதுக்கு எட்டாதவன் என்று ஆத்திகன் கூறுகின்றான். இந்த மூன்றுக்கும் எட்டாதவன் பிறகு எதற்குத்தான் எட்டுவான்? ( விடுதலை - 26.09.1965 )
#MyPeriyar வர்ண பேதம் ஒழிந்த, சாதி முறை, வகுப்பு முறை ஒழிந்த, மதப் பிரிவினைத் தத்துவம் ஒழிந்த எந்தச் சமுதாயமும் - அது தனித்திருந்தாலும், ஒன்றியிருந் தாலும் அதைத் திராவிடச் சமுதாயம் என்று ஒப்புக் கொள்ளவும் நான் தயாராக இருக்கிறேன்.(விடுதலை  -17.11.1947 )
#MyPeriyar அறிவு இருப்பவர்களுக்கு மதம் கிடையாது, அறிவு இருப்பவர்களுக்குக் கடவுள் கிடையாது, அறிவு உள்ளவனுக்கு ஜாதி கிடையாது. ( விடுதலை - 26.10.1960 )
#MyPeriyar இயன்ற உதவியினைச் செய்ய வேண்டும். எந்தக் காரியத்தை முன்னிட்டும் பிறருக்குத் தொந்தரவு செய்யாமல் நடந்து கொள்ள வேண்டும். மற்றவன் உனக்கு எதைச் செய்ய வேண்டும் என்று எதிர்பார்க்கிறாயோ, அதனை நீ மற்றவர்களுக்குச் செய்ய வேண்டும்.~27.8.1971ம் தேதி மணப்பாறையில் ஆற்றிய உரை.
#MyPeriyar தமிழன் தன்னை இந்து என்று கருதியதால் தனது மானத்தையும், ஞானத்தையும் பகுத்தறிவையும், உரிமையையும் இழந்தான்.” ( குடிஅரசு 17.9.1939 )

#MyPeriyar கற்புக்காகக் கணவனின் மிருகச் செயலையும் பொறுத்துக் கொண்டிருக்க வேண்டும் என்ற கொடுமை ஒழிய வேண்டும்.(கு.8.1.28)
#MyPeriyar நமது இலக்கியங்கள் யாவும் நியாயத்திற்காக ஒழுக்கத்திற்காக எழுதப்பட்டிருந்தால் பெண்களுக்கு என்னென்ன நிபந்தனை வைத்திருக்கின்றோமோ அவ்வளவு நிபந்தனைகளை ஆண்களுக்கும் வைத்திருக்க வேண்டுமல்லவா?(வி.1.6.68)
#MyPeriyar ஆண்கள், பெண்களின் விடுதலைக்குப் பாடுபடுவதால் பெண்களின் அடிமைத்தனம் வளருவதுடன், என்றும் விடுதலை பெறமுடியாத கட்டுப்பாடுகள், பலப்பட்டுக் கொண்டு வருகின்றன, பெண்களுக்கு மதிப்புக் கொடுப்பதாகவும், பெண்கள் விடுதலைக்குப் பாடுபடுவதாகவும் ஆண்கள் காட்டிக் கொள்வதெல்லாம் பெண்களை ஏமாற்றுவதற்குச் செய்யும் சூழ்ச்சிகளே ஒழிய வேறல்ல. எங்காவது நரிகளால் ஆடு கோழிகளுக்கு விடுதலை உண்டாகுமா? எங்காவது பூனைகளால் எலிகளுக்கு விடுதலை கிடைக்குமா? எங்காவது முதலாளிகளால் தொழிலாளிகளுக்கு விடுதலை உண்டாகுமா?(கு.12.8.28)
#MyPeriyar பெண்களை வீட்டுவேலை செய்வது, கோலம் போடுவது, சாணி தட்டுவது, பாத்திரம் கழுவுவது, கும்மியடிப்பது, கோலாட்ட மடிப்பது என்பது போன்ற அடிமை வேலைக்குத் தயார் செய்யாதீர்கள்.(கு.8.3.36)
#MyPeriyar இன்றையப் பெண்கள், வெறும் அலங்காரத்தோடு திருப்தியடைந்து விடுகிறார்கள், அல்லது திருப்தி செய்யப்பட்டு விடுகிறார்கள். எனவே இவர்களுக்கு விடுதலை வேட்கை பிறப்பது அரிதாயிருக்கிறது.(வி.26.5.58)
#MyPeriyar கட்சிகள் இந்த நாட்டில் பெரும்பாலும் சாதி இனத்தைப்பற்றியவைகளாக இருப்பதால், பொதுமக்கள் நலத்தைவிட அவரவர்கள் கட்சி நலத்தையே கருதி அரசியல் நடக்கிறது.(கு.18.12.43)
#MyPeriyar நோய் வந்தபின் நோய்க்கு மருந்து கொடுத்து வைத்தியர்கள் குணமாக்குகிறார்கள், ஆனால் அந்த நோய் அடுத்தடுத்து வராமலிருக்க அதற்குரிய காரணங்களைக் கண்டறிந்து அவற்றை ஒழிக்கவேண்டாமா? நோய் வந்துகொண்டேயிருப்பதும் அவ்வப்போது மருந்து கொடுத்துக் கொண்டேயிருப்பதும் பயனுள்ள செயலாகுமா? அதுபோலத்தான் சமுதாயத்தை நாசப்படுத்திவரும் சாதிநோய்க்கான மூலகாரணங்களைக் கண்டறிந்து ஒழிக்க வேண்டும்.(வி.25.7.62)
#MyPeriyar மனிதன் திருடுகிறான், பொய் பேசுகிறான், பாடுபடாமல் வயிறு வளர்க்கப் பார்க்கிறான். இவனை மக்கள் இகழ்வதில்லை. சாதியைவிட்டுத் தள்ளுவதில்லை. ஆனால் சாதியைவிட்டுச் சாதி சாப்பிட்டால் கல்யாணம் செய்தால் சாதியை விட்டுத் தள்ளிவிடப்படுகிறான். இந்த மக்களின் ஒழுக்கம், நாணயம் எப்படிப்பட்டது பாருங்கள்.(கு.20.12.36)
#MyPeriyar ஓரிடத்தில் உப்பு நீர்க் கிணறும், நல்ல தண்ணீர்க்கிணறும் உள்ளது என்றால் நல்ல தண்ணீரை ஒரு பகுதி அனுபவிக்கவேண்டும். உப்புத் தண்ணீரை மற்ற பகுதி மக்கள் அனுபவிக்க வேண்டும். இவர்கள் நல்ல தண்ணீரை உபயோகிக்க லாயக்கற்றவர்கள் என்றிருக்குமானால் அக்கொடுமை எவ்வளவு வேதனை தரக்கூடியது என்பதைச் சிந்திக்க வேண்டும். அப்பேர்ப்பட்ட வேதனைதரும் அளவுக்குச் சாதிமுறைகள் அமைக்கப்பட்டிருக்கின்றன. ஒரு சிலர் மட்டும் சுகம் அனுபவிப்பதற்கென்றும் மற்ற பலர் வேதனைப்டுவதற்கென்றுமே அமைக்கப்பட்ட சாதிமுறைகள் இந்நாட்டை விட்டு அகலும்வரை நமக்குள்ள கொடுமைகள் நீங்காதென்பது திண்ணம்.(வி.30.3.56)
#MyPeriyar சாதி ஒழிப்புக்கு இன்றைய அரசியல் சட்டம் இடம் தரவில்லை. அதை, அடிப்படை உரிமைக்கு விரோதம் என்கிறது. அதேபோல்தான் வகுப்புவாரி விகிதப் பேச்சும் பேசாதே, அது வகுப்புத் துவேஷம் என்கிறது. சாதி இருப்பது தவறல்லவாம். சாதிப்படி உரிமை கேட்டால் மட்டும் தவறு என்றால் இதைவிடப் பித்தலாட்டம் வேறு இருக்க முடியுமா?(வி.24.5.61)
